# Elaneeru Homestay — website

Marketing site for Elaneeru Homestay in Chikkamagaluru (React, Vite, Tailwind, shadcn-style UI).

## Scripts

- `npm run dev` — local dev server (port 8080)
- `npm run build` — production build
- `npm run preview` — preview the build
- `npm run test` — Vitest unit tests

## E2E (Playwright)

Add specs under `e2e/`, then run `npx playwright test` (uses `playwright.config.ts` and starts the dev server when needed).
